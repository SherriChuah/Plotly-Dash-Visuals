# Plotly-Dash-Visuals
Experimenting with Plotly and Dash to create a dashboard

## To run program
1. Create a virtual environment
```asciidoc
python3 -m venv {venv_name}
```
2. Activate the virtual environment
```asciidoc
source {venv_name}/bin/activate 
```
3. Run the app
```asciidoc
python app.py
```
